    So what is smartTips?

    smartTips is a jQuery plugin that finds words on your webpage that you define. Then it wraps them in a span with a class of .smartTip. When you hover over the span it will pop out a tooltip with the corresponding text.

    How cool is that?

    How does it work?

    It's deceptively simple. To get started, all you have to do is:

    1. get the smartTips repo
    2. include the smartTips.js file in your project
    3. open the smartTips.js file
    4. enter 'word' : 'definition' under 'var definitions'

    var definitions = {
  lorem : "A mark of class",
  tooltips : "Messages that appear when a cursor is positioned over a term, icon, image, hyperlink, or other element in a graphical user interface.",
  smarttips: "smartTips is not a registered trademark.",
  'web project': "You just have to include the smartTips.js script"
}
